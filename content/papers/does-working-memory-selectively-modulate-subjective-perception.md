## Abstract Snapshot
This study aimed to investigate how working memory interacts with perception at the behavioral level. We examined whether working memory modulates subjective perception when the general capacity for perceptual processing is kept constant. This possibility is in part motivated by the neuropsychological phenomenon of blindsight, in which subjective perception and general perceptual capacity are dissociated. In a previous pilot study (n = 8), we manipulated the congruency between working memory and perceptual content. We observed a trend suggesting that holding categorically incongruent content in working memory selectively impaired sensitivity (da) in a Yes/No (Y/N) detection task, even when perceptual processing capacity was matched using a two-interval forced choice (2-IFC) task. In the current study (n = 16), we tested the effect of congruency on the Y/N discrimination task with an increased sample size. To match perceptual processing capacity across conditions, we titrated stimulus contrast in a two-interval forced choice (2-IFC) task again. Contrary to the previous pilot study, we did not find evidence that working memory selectively modulates subjective perception, as measured by sensitivity (da) in the Y/N task, when processing capacity in the 2-IFC task was matched. Given this discrepancy, we pre-register a follow-up data collection on the original pilot design (i.e., Y/N detection), including the analysis plan and stopping rule.
